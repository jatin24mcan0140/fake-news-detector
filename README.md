# 🔍 Fake News Detector

An end-to-end fake news detection system using BERT and Machine Learning models achieving **99.41% accuracy**.

## 📊 Models Used
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 95.77% |
| Decision Tree | 99.64% |
| Random Forest | 99.61% |
| Gradient Boosting | 99.60% |
| **BERT (Deep Learning)** | **99.41%** |

## 📁 Dataset
- **WELFake Dataset** — 72,134 news articles
- Source: Kaggle

## 🛠️ Tech Stack
- Python, PyTorch, HuggingFace Transformers
- BERT (bert-base-uncased)
- Scikit-learn, NLTK, TF-IDF
- Streamlit

## ⚙️ Installation
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📌 Features
- ✅ Real-time fake news detection
- ✅ Confidence score with progress bar
- ✅ Risk level classification
- ✅ BERT context-aware detection
